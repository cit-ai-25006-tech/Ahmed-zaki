# Student Grading System

## About the Project
This is a simple C++ program that helps store and display a student's academic information. It collects the student's name, their department, and their grades for 6 different subjects to calculate their total score and average.

## Key Features
- **Student Data**: Stores the student's name and department.
- **Grade Input**: Easily enter grades for 6 subjects.
- **Auto Calculation**: Automatically calculates the total score out of 600 and the average percentage.

## How to Run
1. Copy the code into your C++ compiler (like Dev-C++, Code::Blocks, or VS Code).
2. Run the program.
3. Enter the requested information when prompted.
